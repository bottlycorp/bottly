## Environments variables

```

BOT_TOKEN="your token"
CLIENT_ID="bot id"
WEBHOOK_URL="a webhook url"

# Database:

# OpenAI :
GITHUB_TOKEN="your openai key"

# Stripe:
STRIPE_PUBLIC_KEY=""
STRIPE_PRIVATE_KEY=""
STRIPE_LIVE_MODE=""

# Miscs:

SUPPORT_GUILD_ID=""
```
